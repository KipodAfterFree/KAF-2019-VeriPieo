This is the developer guide for VeriPieo devices;

Structure of a VP app:

app.html - layout of the app.

app.js - js code of the app.

app.png - icon of the app.

the html file will be loaded before the js file.

the js file decides when to show the layout - using the app_loaded() function.

Name of command Parameters Explanation
app_loaded None this function switches the user's view to the app's view
app_write key, value, callback writes the given value to the given key in persistent server memory. - calls callback when done
app_read key reads the value from the given key, from persistent server memory. - calls callback with the value when done
device_app appId, callback loads the app (appId) on the device, then evals callback.
device_popup text, timeout, color, onclick pops up a popup

HTML Tags:

Name Explanation
back holds the color tags, for device background color
color contains the color for the background gradient
